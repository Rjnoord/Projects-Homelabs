# Cloud Labs

This section groups my cloud security and cloud administration labs.

- [azure-key-vault-secrets-management-lab/](azure-key-vault-secrets-management-lab/)  
  Azure Key Vault lab focused on secrets management across deployment, secret creation, versioning and rotation, deletion and recovery awareness, and audit log review.

- [AWS-IAM-CloudTrail-Least-Privilege-Lab](https://github.com/Rjnoord/AWS-IAM-CloudTrail-Least-Privilege-Lab)  
  AWS lab focused on IAM group design, user provisioning, admin versus least-privilege comparison, and CloudTrail event review.

- [Oracle-oci-secure-vm-lab](https://github.com/Rjnoord/Oracle-oci-secure-vm-lab)  
  Oracle Cloud Infrastructure lab covering Linux VM deployment, SSH-key-based access, and network-level security controls.
