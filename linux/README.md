# Linux Labs

This section groups my Linux-focused homelab projects.

- [Linux-SSH-Hardening-Brute-Force-Mitigation-Lab-Fail2Ban-](https://github.com/Rjnoord/Linux-SSH-Hardening-Brute-Force-Mitigation-Lab-Fail2Ban-)  
  Linux hardening lab demonstrating SSH brute-force mitigation using Fail2Ban and defensive configuration practices.

- [ssh-bruteforce-detection-network-analysis](https://github.com/Rjnoord/ssh-bruteforce-detection-network-analysis)  
  SOC-style home lab focused on identifying and analyzing unauthorized SSH access attempts using Linux and network traffic correlation.

- [Linux-Router-Configuration-Lab](https://github.com/Rjnoord/Linux-Router-Configuration-Lab)  
  Linux networking lab demonstrating Layer 3 routing between isolated networks through manual routing configuration.

- [linux-static-ip-subnet-lab](https://github.com/Rjnoord/linux-static-ip-subnet-lab)  
  Networking lab focused on subnetting, static IP configuration, CIDR planning, and scoped host discovery validation.

- [Oracle-oci-secure-vm-lab](https://github.com/Rjnoord/Oracle-oci-secure-vm-lab)  
  Oracle Cloud Infrastructure lab covering Linux VM deployment, SSH-key-based access, and network-level security controls.
