# Projects-Homelabs

## Overview

This repository is a public portfolio index of my cybersecurity, networking, Linux, and SIEM home lab projects. Each project is maintained in its own standalone repository so that the documentation, screenshots, configurations, and write-ups stay focused and easy to review.

The purpose of this repository is to give recruiters, hiring managers, and technical reviewers one place to quickly understand the scope of my hands-on work.

## What This Portfolio Shows

- Linux administration and system hardening
- Splunk SIEM setup, log ingestion, and detection workflows
- Wireshark packet analysis and protocol visibility
- Nmap reconnaissance and network analysis
- Routing, subnetting, and network configuration
- Cloud VM security fundamentals
- Security awareness and phishing simulation labs
